# LoL Champions App - Project Description

## English Version

### Project Overview
**LoL Champions App** is a personal, non-commercial project developed for educational purposes and personal use only. The application runs exclusively on a local environment (localhost) and is not intended for public deployment or distribution.

### Purpose
The primary goal of this project is to provide a simple, personal tool to:
- Browse League of Legends champion information (abilities, lore, skins)
- View in-game item details and statistics
- Consult personal match history using the Riot ID (Game Name + Tag Line)

### Technical Details
- **Backend:** Spring Boot (Java)
- **Frontend:** Angular
- **Data Sources:** 
  - Data Dragon API (for static game data: champions, items, images)
  - Riot Games API (for match history consultation)
- **Environment:** Local development only (localhost)
- **Users:** Single user (myself)

### Non-Commercial Statement
This project:
- ✅ Is developed purely for personal learning and educational purposes
- ✅ Runs only on my local machine
- ✅ Will NOT be deployed to any public server
- ✅ Will NOT generate any revenue or be monetized in any way
- ✅ Will NOT collect or store any third-party user data
- ✅ Follows all Riot Games API Terms of Service and policies
- ✅ Properly attributes all data and assets to Riot Games

### Security Compliance
- API keys are stored in environment variables, never hardcoded
- API key is not shared or exposed in source code
- All API calls are made via HTTPS
- The application follows the new Riot ID system (Game Name + Tag Line) instead of deprecated Summoner Names

### Contact
This project is maintained by a single developer for personal use. It is not associated with any organization, company, or commercial entity.
